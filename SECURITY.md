# Security Policy

## Reporting

Please report security issues with GitHub private vulnerability reporting:

https://github.com/AtharvaMaik/donecheck/security/advisories/new

If private reporting is unavailable, open an issue with a minimal description and omit exploit details until a maintainer responds.

## Scope

Supported versions are the latest GitHub release and `main`.

DoneCheck is a local CLI and GitHub Action. It does not collect telemetry, call external services, or upload repository contents.
